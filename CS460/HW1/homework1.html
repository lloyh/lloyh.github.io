In this assignment we learned the coding essentials of HTML5, CSS and Bootstrap. We also learned to use Git source control via Git Bash. This is an important skill to learn as it allows multiple software developers to work simultaneously on the same source code. Git also allows to revert changes to the code when needed.

The website content and its webpages

I decided to make my website about Tesla Motors. From the navigation bar, the “Home” link will be directing to the index page that presents the assignment requirements. The “Tesla Motors” link will direct to the page that shows a brief introduction to all Tesla vehicles with text and images to each vehicle and will link each car to its corresponding webpage. The “Contact” link will direct to a webpage that contains the company’s general phone numbers and email information. The “About” link will link to a page that presents a picture of the current production vehicles along with general information about electric vehicles and facts about each vehicle.

From the tesla_lineup.html page (accessed via the “Tesla Motors” navigation link) each vehicle picture and corresponding model name will be links to their corresponding pages with brief information and other pictures of the same vehicle. I placed each image inside of div elements which in turn each one is part of an unordered list. This meets one of the assignment requirements.
